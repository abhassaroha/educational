# Educational Visualisations

This repo includes visualisations i have made in [Three.js](https://github.com/mrdoob/three.js) in response to the daily curiosities of my 5 year old daughter's growing mind. A lot of the code here is generated with aid of AI (One shot with somewhat detailed prompts).

Over time this may grow with visualisations of some of my own curiosities.


## Why Three.js

The seed for growing with [`Three.js`](https://github.com/mrdoob/three.js) was a demo I saw online of how someone could single shot a solar system visualisation. That is precisely what my daughter had been learning at that time. So that set the path I took in terms of tech stack. 

But I am not tied down to [`Three.js`](https://github.com/mrdoob/three.js) and may explore other frameworks in future. React Three js hasn't seemed that straightforward to me, so probably I may phase out the React part and start from something simpler. I find myself struggling (when not asking AI to fix issues) with abstraction layers of React Three. This is not the age to be bogged down by frameworks!
